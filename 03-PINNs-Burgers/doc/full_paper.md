# Physics-Informed Neural Networks による Burgers 方程式の順問題・逆問題への応用

---

## Abstract

本稿では，Physics-Informed Neural Networks（PINNs）を用いて
1次元粘性 Burgers 方程式の順問題および逆問題を解く手法を提案する．
PINNs は，ニューラルネットワークの損失関数に支配方程式の残差を組み込むことにより，
少量の境界・初期条件データのみから物理的に整合した解を学習する．
順問題では任意の時空間点における速度場 `u(t, x)` を予測し，
逆問題では観測データから未知の動粘性係数 `ν` を同定する．
数値実験により，本手法が参照解に対して相対 L2 誤差 `O(10^{-3})` 程度の
精度で解を再現できることを示す．

---

## 1. Introduction

偏微分方程式（PDE）は流体力学，熱移動，量子力学など
広範な物理現象の記述に用いられる．
従来の数値解法（有限差分法，有限要素法など）はメッシュ生成を必要とし，
複雑な形状や高次元問題における計算コストが課題となる．

近年，深層学習を科学計算に応用する試みが活発化しており，
Raissi et al. (2019) は PINNs を提案した．
PINNs は支配方程式を損失関数として直接組み込むことで，
メッシュフリーかつデータ効率の高い PDE ソルバーとして機能する．

本稿では，PINNs の理解と検証のための標準的なベンチマーク問題として
1次元粘性 Burgers 方程式を採用する．
Burgers 方程式は移流と拡散の非線形競合を含み，
衝撃波（ショック）の形成という物理的に重要な現象を示す一方，
解析解が既知であるため，数値精度の検証が厳密に行える利点がある．

---

## 2. Problem Formulation

### 2.1 支配方程式

1次元粘性 Burgers 方程式を以下のように定義する：

```
∂u     ∂u        ∂²u
── + u ── = ν ────       (1)
∂t     ∂x        ∂x²

  x ∈ Ω  = [-1, 1]
  t ∈ [0, T] = [0, 1]
```

ここで `u(t, x)` は流体速度場，`ν > 0` は動粘性係数である．
左辺第2項 `u ∂u/∂x` は非線形移流項であり，速度の速い領域が
遅い領域に追いつくことで波形の急峻化（衝撃波形成）を引き起こす．
右辺 `ν ∂²u/∂x²` は粘性拡散項であり，速度場を空間的に平滑化する方向に作用する．
`ν` が小さいほど衝撃波は鋭く，`ν` が大きいほど速やかに拡散する．

### 2.2 初期・境界条件

```
u(0, x)   = -sin(πx),    x ∈ [-1, 1]     (初期条件)    (2)
u(t, -1)  =  0,           t ∈ [0, 1]     (左境界条件)   (3)
u(t,  1)  =  0,           t ∈ [0, 1]     (右境界条件)   (4)
```

本設定では `ν = 0.01/π` とする．
この小さな粘性係数により，`t ≈ 0.4` 付近で衝撃波が形成され，
古典的数値解法に対して挑戦的な問題設定となる．

## 3. Methodology

### 3.1 Physics-Informed Neural Networks

PINNs では，未知の解 `u(t, x)` をニューラルネットワーク
`u_θ(t, x)` によって近似する．ここで `θ` はネットワークのパラメータ全体を表す．
ネットワークは時空間座標 `(t, x) ∈ R²` を入力として受け取り，
スカラー値 `u ∈ R` を出力する全結合型フィードフォワードネットワークである．

`L` 個の隠れ層と各層の幅 `N` を持つネットワークを以下のように定義する：

```
u_θ(t, x) = (W_L ∘ σ ∘ W_{L-1} ∘ σ ∘ … ∘ σ ∘ W_1)([t, x]^T)    (5)

  W_l  : l 番目の層の重み行列およびバイアスベクトル
  σ    : 活性化関数（本稿では tanh を採用）
  ∘    : 関数合成
```

活性化関数として `tanh` を採用する理由は，
`tanh` が無限回連続微分可能（C∞ 級）であり，
PDE に含まれる高階偏微分の計算において数値的に安定した結果を与えるためである．

### 3.2 PDE 残差の定義

ネットワーク出力 `u_θ(t, x)` に対して，Burgers 方程式の残差を：

```
f(t, x; θ) := ∂u_θ/∂t  +  u_θ · ∂u_θ/∂x  -  ν · ∂²u_θ/∂x²    (6)
```

と定義する．真の解が `u_θ` によって近似されているならば，
`f(t, x; θ) = 0` がドメイン内のすべての点で成立するはずである．
式 (6) に現れる偏微分 `∂u_θ/∂t`，`∂u_θ/∂x`，`∂²u_θ/∂x²` は
自動微分（automatic differentiation）により解析的に計算される．
この操作は，連鎖律を計算グラフに沿って逆方向に適用することで実現され，
数値微分と異なり離散化誤差を含まない．

### 3.3 損失関数

PINNs の損失関数は，データ損失 `L_data` と物理損失 `L_phys` の和として定義される：

```
L(θ) = L_data(θ) + L_phys(θ)    (7)
```

**データ損失**は初期・境界条件の点集合
`{(t_u^i, x_u^i, u^i)}_{i=1}^{N_u}` 上で定義される：

```
         1   N_u
L_data = ─── Σ   | u_θ(t_u^i, x_u^i) - u^i |²              (8)
         N_u i=1
```

**物理損失**はコロケーション点集合
`{(t_f^j, x_f^j)}_{j=1}^{N_f}` 上で定義される：

```
          1   N_f
L_phys =  ─── Σ   | f(t_f^j, x_f^j; θ) |²                  (9)
          N_f j=1
```

コロケーション点は観測ラベルを必要とせず，
時空間ドメイン `[-1,1] × [0,1]` 上の点座標のみで構成される．
したがって，式 (9) は PDE そのものを「教師信号」として利用しており，
物理法則を正則化項として学習に組み込む役割を果たす．

### 3.4 訓練手順

パラメータ `θ` は損失 `L(θ)` を最小化することで求められる：

```
θ* = argmin L(θ)    (10)
      θ
```

最適化は Adam オプティマイザによる一次勾配法を用いる．
精度をさらに高める場合には，Adam による粗い最適化の後，
L-BFGS による準ニュートン法を適用することが推奨される（Raissi et al., 2019）．

## 4. Algorithms

### Algorithm 1: PDE 残差の計算

PDE 残差 `f(t, x; θ)` の計算は，ネットワーク出力に対する自動微分を
連続的に適用することで実現される．以下にその手順を示す．

```
Algorithm 1: Computation of PDE Residual f(t, x; θ)
─────────────────────────────────────────────────────────────────
Input : コロケーション点集合  {(t_f^j, x_f^j)}_{j=1}^{N_f}
        学習済みパラメータ θ，動粘性係数 ν
Output: PDE 残差ベクトル  f ∈ R^{N_f}
─────────────────────────────────────────────────────────────────
1:  t, x ← {(t_f^j, x_f^j)} を計算グラフに登録
           （自動微分の対象として宣言）

2:  u ← u_θ(t, x)
           ネットワークの順伝播により u(t, x) を計算

3:  u_t ← ∂u/∂t
           u を t に関して自動微分（1 階）
           ※ 後続の微分のために計算グラフを保持する

4:  u_x ← ∂u/∂x
           u を x に関して自動微分（1 階）
           ※ 後続の微分のために計算グラフを保持する

5:  u_xx ← ∂u_x/∂x
           u_x を x に関して自動微分（2 階微分に相当）

6:  f ← u_t  +  u · u_x  -  ν · u_xx
           Burgers 方程式の残差を構成（式 (6) に対応）

7:  return f
─────────────────────────────────────────────────────────────────
```

Step 3–5 における「計算グラフの保持」とは，
`u_t` や `u_x` 自体をさらに微分できるよう，
それらを導出した計算経路を破棄しないことを意味する．
この操作により，`u_xx` のような 2 階微分が連続した自動微分として実現される．

---

### Algorithm 2: PINN 順問題ソルバー

```
Algorithm 2: PINN Forward Problem Solver
─────────────────────────────────────────────────────────────────
Input : 境界・初期条件データ  D_u = {(t_u^i, x_u^i, u^i)}_{i=1}^{N_u}
        コロケーション点      D_f = {(t_f^j, x_f^j)}_{j=1}^{N_f}
        動粘性係数 ν（既知），学習率 η，最大エポック数 E_Adam, E_LBFGS
Output: 最適化されたパラメータ θ*
─────────────────────────────────────────────────────────────────
1:  θ ← Xavier 一様初期化によりネットワーク重みを初期化

2:  optimizer_1 ← Adam(θ, lr=η)
    optimizer_2 ← L-BFGS(θ)

    ── Phase 1: Adam による粗い最適化 ──────────────────────────

3:  for e = 1, 2, …, E_Adam do

4:      u_pred ← u_θ(t_u, x_u)
                  境界・初期条件点における順伝播

5:      L_data ← (1/N_u) Σ_i | u_pred^i - u^i |²
                  データ損失を計算（式 (8)）

6:      f ← Algorithm 1(t_f, x_f, θ, ν)
                  コロケーション点における PDE 残差を計算

7:      L_phys ← (1/N_f) Σ_j | f^j |²
                  物理損失を計算（式 (9)）

8:      L ← L_data + L_phys
                  総損失を計算（式 (7)）

9:      θ ← θ - η · ∇_θ L
                  誤差逆伝播により勾配を計算し，パラメータを更新

10: end for

    ── Phase 2: L-BFGS による精緻化 ────────────────────────────

11: for e = 1, 2, …, E_LBFGS do

12:     L ← L_data + L_phys   （手順 4–8 と同様）

13:     θ ← θ - H^{-1} ∇_θ L
                  近似ヘッセ行列 H の逆行列を用いたパラメータ更新

14: end for

15: return θ*
─────────────────────────────────────────────────────────────────
```

Phase 1 の Adam は広い損失景観を効率よく探索し，
Phase 2 の L-BFGS は局所的な曲率情報を活用して高精度な極小点に収束する．
この二段階最適化は Raissi et al. (2019) で採用された標準的な手順である．

---

### Algorithm 3: PINN 逆問題ソルバー（動粘性係数の同定）

```
Algorithm 3: PINN Inverse Problem Solver
─────────────────────────────────────────────────────────────────
Input : 観測データ  D_obs = {(t^i, x^i, u^i)}_{i=1}^{N_obs}
        コロケーション点  D_f = {(t_f^j, x_f^j)}_{j=1}^{N_f}
        ν の初期推定値 ν_0，学習率 η，最大エポック数 E
Output: 最適化されたパラメータ θ*，推定された動粘性係数 ν*
─────────────────────────────────────────────────────────────────
1:  θ  ← Xavier 一様初期化
    ν  ← ν_0
           ν を θ と並ぶ学習可能変数として宣言する

2:  optimizer ← Adam([θ, ν], lr=η)
                  θ と ν を同一のオプティマイザで更新する

3:  for e = 1, 2, …, E do

4:      u_pred ← u_θ(t_obs, x_obs)

5:      L_data ← (1/N_obs) Σ_i | u_pred^i - u^i |²

6:      f ← u_t  +  u · u_x  -  ν · u_xx
                  ν が学習変数であるため，∂f/∂ν = -u_xx が
                  自動微分の計算グラフに自動的に含まれる

7:      L_phys ← (1/N_f) Σ_j | f^j |²

8:      L ← L_data + L_phys

9:      (θ, ν) ← (θ, ν) - η · ∇_{(θ,ν)} L
                  θ と ν が同時に更新される

10: end for

11: return θ*, ν*
─────────────────────────────────────────────────────────────────
```

Algorithm 3 が Algorithm 2 と異なる点は Step 1 と Step 9 のみである．
`ν` を学習可能変数として宣言することで，
勾配 `∂L/∂ν` が自動微分によって自然に得られ，
ネットワーク重みの更新と同一のループ内で `ν` が逐次修正される．

---

## 5. Inverse Problem: 動粘性係数の同定

### 5.1 問題設定

逆問題では `ν` を未知量として扱い，
時空間ドメイン内の観測データ `{(t^i, x^i, u^i)}_{i=1}^{N_obs}` から
`ν` を同定することを目的とする．

### 5.2 定式化

逆問題における損失関数は順問題の式 (7) と同一形式を保つ．
ただし `ν` をネットワークパラメータ `θ` と同時に学習される変数として扱う：

```
(θ*, ν*) = argmin L(θ, ν)    (11)
             θ, ν
```

PINNs の逆問題への適用が優れている点は，
PDE の残差 (6) が `ν` について線形であるため，
勾配 `∂L/∂ν` が自動微分によって自然に計算される点にある．
これにより，ネットワークの重み更新と物理パラメータの同定が
単一の最適化ループ内で同時に実行される（Algorithm 3 参照）．

## 6. 数値実験

### 6.1 実験設定

```
対象方程式  : 1次元粘性 Burgers 方程式 (1)
動粘性係数  : ν = 0.01/π
ドメイン    : x ∈ [-1, 1],  t ∈ [0, 1]
初期条件    : u(0, x) = -sin(πx)

ネットワーク構造  : 入力層(2) + 隠れ層 4 層(各 20 ニューロン) + 出力層(1)
活性化関数        : tanh
重み初期化        : Xavier 初期化

データ点数    N_u : 100    （初期・境界条件）
コロケーション点数 N_f : 10,000
オプティマイザ    : Adam → L-BFGS（二段階）
```

### 6.2 評価指標

予測精度の評価には相対 L2 誤差を用いる：

```
         || u_pred - u_ref ||_2
e_rel =  ─────────────────────      (12)
            || u_ref ||_2
```

ここで `u_ref` は解析解または高精度数値解（参照解）である．
論文の再現性確認の基準として `e_rel < 1.0 × 10^{-2}` を目標とする．

### 6.3 期待される結果

順問題においては，ネットワークが初期の正弦波プロファイルから
`t ≈ 0.4` における衝撃波形成，さらに `t → 1` での拡散による平滑化という
一連のダイナミクスを再現することが期待される．

逆問題においては，観測データが十分に分布していれば，
真値 `ν = 0.01/π ≈ 3.183 × 10^{-3}` に対して
数パーセント以内の誤差で `ν*` が収束することが期待される．

## 7. Discussion

PINNs の本質的な利点は，支配方程式が既知であれば
ラベルなしのコロケーション点を大量に使用できる点にある．
これは，実験計測コストが高い材料科学・生体力学などの分野において，
少ない観測データから物理的に整合した予測を行う上で特に有効である．

一方で，PINNs には以下の課題が知られている：

```
(1) 複数の損失項の重み付けの難しさ
    L_data と L_phys のスケールが異なる場合，
    一方の損失が支配的になり学習が不安定になることがある．

(2) 衝撃波など急峻な解への対応
    ν が非常に小さい場合，解が急峻になり
    コロケーション点の均一サンプリングでは残差が過小評価される．
    適応的コロケーション点再サンプリングが有効な対策として知られる．

(3) 順問題における計算コスト
    従来の有限差分法と比較して，PINNs は順問題では
    必ずしも計算効率に優れるわけではなく，
    逆問題やパラメータ同定での強みが際立つ．
```

---

## 8. Conclusion

本稿では，PINNs を用いた Burgers 方程式の順問題・逆問題への適用を述べた．
PINNs は支配方程式を損失関数として自然に組み込むことで，
メッシュフリーかつデータ効率の高い物理シミュレーションを実現する．
逆問題においては，ネットワーク重みと物理パラメータを同一の最適化ループで
同定できる点が特に有用である．
本手法は Burgers 方程式を超えて，Navier-Stokes 方程式や熱方程式など
より複雑な PDE への拡張性を持つ．
